# Top_Songs_Analysis
Data Analytics Bootcamp 2021 Final Project

Solo Project

Topic: Spotify Charts Analysis (Top 200 Charts 2020-present)

Questions: 
- What songs topped charts the most?
- What songs have the highest # of streams?
- Artists with the most followers?
- Did the top songs tend to have similarities in any features (such as danceability, energy, etc.)
- Model: Can we predict whether a song has hit #1 (Yes/No) based on the data provided.
